About This Project

This image serach engine is the final course project for Web Information Retrieval in 2018/5

Details

This image search engine is based on Web technology and deep learning technology. There three main functions:

  • Image serach, find similar images for users
  • Image classification, tell the user which class the uploaded query image belongs to
  • Image caption, give general discription of the images
  • Image editting, a simple UI for editing the images
